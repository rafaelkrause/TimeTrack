Quick install

Two clicks on Windows, one line on Linux / macOS.

  1. Download the installer — TimeTrack-Setup.exe Releases
  2. Run it. If SmartScreen blocks it, click More info → Run anyway (installer is not yet code-signed).
  3. Done. The UI opens at http://localhost:5000.

Per-user install (no UAC). Embedded Python. Data at %APPDATA%\TimeTrack, preserved across updates.

One command does it all: downloads the wheel, creates an isolated virtualenv and drops the launcher on your PATH.

curl -fsSL https://raw.githubusercontent.com/rafaelkrause/TimeTrack/main/install-remote.sh | bash

Needs Python 3.10+, curl and bash. Append -s -- --service for autostart (systemd on Linux, LaunchAgent on macOS).

For development or running before a release has been published:

git clone https://github.com/rafaelkrause/TimeTrack.git
cd TimeTrack
./install.sh
./timetrack.sh
