Start, pause, resume, stop. Bandeja do sistema controla tudo sem abrir o navegador.

% do turno, timeline do dia, visões de semana e mês. Seu progresso real, não o planejado.

Instalação rápida

Dois cliques no Windows, uma linha no Linux / macOS.

  1. Baixe o instalador JobTracker-Setup.exe Releases
  2. Execute. Se o SmartScreen avisar, clique Mais informações → Executar assim mesmo (instalador ainda não assinado).
  3. Pronto. A UI abre em http://localhost:5000.

Instalação por usuário (sem UAC). Python embutido. Dados em %APPDATA%\JobTracker, preservados entre atualizações.

Um comando faz tudo: baixa o wheel, cria um virtualenv isolado e registra o launcher no PATH.

curl -fsSL https://raw.githubusercontent.com/rafaelkrause/job_tracker/main/install-remote.sh | bash

Requer Python 3.10+, curl e bash. Adicione -s -- --service para autostart (systemd no Linux, LaunchAgent no macOS).

Para desenvolver ou rodar antes de uma release publicada:

git clone https://github.com/rafaelkrause/job_tracker.git
cd job_tracker
./install.sh
./job-tracker.sh
