# Screenshots

Drop PNG/JPG screenshots here. They are referenced from the top-level `README.md` and from `docs-src/index.*.md` / `docs-src/user-guide.*.md`.

Expected filenames (referenced in the docs):

| File | Content |
|---|---|
| `dashboard.png` | Main dashboard: current activity, timeline, shift progress, target. |
| `settings.png` | Settings page: shifts, target, theme, user name, phrases toggle. |
| `export.png` | Export page / modal: date range picker + CSV/TSV download. |
| `tray.png` | System-tray icon with pause/resume/stop menu open. |

Recommended size: 1280×800 (or 2560×1600 for retina). Crop tight to the app window.
