Instalação

Job Tracker roda localmente como um servidor web em http://localhost:5000. Escolha o método que melhor se encaixa no seu ambiente.

Pré-requisitos

  • Linux / macOS: Python 3.10+, pip, git, bash.
  • Windows: nada — o instalador NSIS traz o Python embutido. Para instalação manual, Python 3.10+.
  • Opcional: pystray + Pillow para o ícone na bandeja do sistema.

Linux / macOS (script)

A forma recomendada é usar o script install.sh:

git clone https://github.com/rafaelkrause/job_tracker.git
cd job_tracker
./install.sh

O script:

  1. Verifica se Python 3.10+ está disponível.
  2. Cria um ambiente virtual em .venv/.
  3. Instala dependências do requirements.txt.
  4. Opcionalmente instala pystray + Pillow para o system tray.
  5. Cria um atalho .desktop (se ambiente gráfico detectado).

Após instalar, execute o app com:

./job-tracker.sh
# ou
python3 run.py

Instalação manual (Linux / macOS)

Se preferir controlar cada passo:

git clone https://github.com/rafaelkrause/job_tracker.git
cd job_tracker

python3 -m venv .venv
source .venv/bin/activate

pip install -r requirements.txt

# opcional: suporte a bandeja do sistema
pip install pystray Pillow

python3 run.py

Use python3 run.py --no-browser para iniciar sem abrir o navegador automaticamente.

Windows — instalador NSIS

Baixe JobTracker-Setup-X.Y.Z.exe da página de releases e execute.

  • Instalação por usuário: não requer UAC. Destino padrão %LOCALAPPDATA%\Programs\JobTracker.
  • Python embutido: versão portátil do Python é empacotada — nada é instalado globalmente.
  • Dados: configuração e histórico ficam em %APPDATA%\JobTracker.
  • Atalhos: desktop e menu Iniciar são opcionais na tela do assistente.
  • Serviço do Windows: componente opcional com NSSM, inicia o tracker no boot (UAC apenas se marcado).

Windows — instalação manual

  1. Instale Python 3.10+. Marque Add Python to PATH.
  2. Baixe o código (ZIP da release ou git clone).
  3. No PowerShell: 
    cd job_tracker
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python run.py

Gerar o instalador do zero

O instalador Windows é gerado em um host Linux com NSIS:

sudo apt install nsis
./installer/build_installer.sh 1.0.0
# saída: installer/JobTracker-Setup-1.0.0.exe

O workflow .github/workflows/build-installer.yml faz isso automaticamente quando uma tag v* é publicada.

Iniciar automaticamente

Linux — systemd (user unit)

Crie ~/.config/systemd/user/job-tracker.service:

[Unit]
Description=Job Tracker
After=network.target

[Service]
Type=simple
WorkingDirectory=%h/job_tracker
ExecStart=%h/job_tracker/.venv/bin/python run.py --no-browser
Restart=on-failure

[Install]
WantedBy=default.target
systemctl --user daemon-reload
systemctl --user enable --now job-tracker.service

Windows — serviço via NSSM

O instalador oficial oferece a opção de instalar como serviço. Alternativamente, após a instalação manual, use o NSSM:

nssm install JobTracker "C:\Caminho\job_tracker\.venv\Scripts\python.exe" run.py --no-browser
nssm start JobTracker

Atualizar

Linux / macOS

cd job_tracker
git pull
source .venv/bin/activate
pip install -r requirements.txt --upgrade

Windows

Baixe e execute o novo JobTracker-Setup-X.Y.Z.exe. O instalador preserva sua configuração e dados em %APPDATA%\JobTracker.

Desinstalar

  • Linux/macOS: rm -rf job_tracker/ e, se aplicável, systemctl --user disable --now job-tracker.service.
  • Windows: Painel de Controle → Programas → Job Tracker → Desinstalar, ou via Configurações → Aplicativos.
  • Para remover também os dados: apague o diretório data/ (Linux) ou %APPDATA%\JobTracker (Windows).

Problemas comuns

Porta 5000 em uso

Altere a porta em config.json:

{ "port": 5050 }

Ou via página /settings.

O ícone da bandeja não aparece

Instale as dependências opcionais:

pip install pystray Pillow

No Linux, em alguns desktops (GNOME puro) é preciso a extensão AppIndicator.

“ModuleNotFoundError: flask”

O venv não está ativo. Rode source .venv/bin/activate (Linux/macOS) ou .venv\Scripts\Activate.ps1 (Windows).

Windows bloqueia o .exe (SmartScreen)

Clique em Mais informações → Executar assim mesmo. Instaladores sem assinatura digital acionam esse aviso.
