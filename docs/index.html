PT / EN

Rastreie suas horas sem fricção

Uma ferramenta leve, local e autônoma de controle de horas para profissionais de agência. Interface web em localhost:5000, armazenamento em JSON, sem banco de dados.

Como instalar Documentação
Python 3.10+ Flask Bootstrap 5 Linux Windows

Recursos principais

Máquina de estados clara

Iniciar, pausar, retomar e finalizar atividades. O tempo de pausa é subtraído automaticamente da duração total.

Dashboard diário

Timeline visual, barra de progresso por turno e porcentagem de meta. Só considera as horas já decorridas.

Armazenamento em JSON

Um arquivo por mês em data/YYYY-MM.json. Sem ORM, sem banco. Fácil de inspecionar à mão.

Turnos configuráveis

Defina turnos por dia da semana com múltiplos blocos. Porta, tema (claro/escuro/auto) e meta também editáveis.

Exportação CSV/TSV

Formatada para colar manualmente no iClips. Exporta apenas atividades finalizadas em um intervalo de datas.

System tray opcional

Pausar, retomar e parar pelo ícone da bandeja do sistema. Requer pystray (fallback gracioso se ausente).

Escrita atômica

Toda gravação usa arquivo temporário + fsync + os.replace(). Seguro contra crashes e queda de energia.

Tema claro / escuro

Automático conforme o sistema operacional, ou fixo. Preferência persistida no localStorage.

Interface em pt-BR

Textos da UI em português brasileiro. Pensado para o fluxo de quem faz apontamento diário em agência.

Instalação rápida

Linux / macOS

Requer Python 3.10+ e git.

git clone https://github.com/rafaelkrause/job_tracker.git
cd job_tracker
./install.sh
./job-tracker.sh

O script cria um .venv, instala dependências e abre o navegador.

Windows

Use o instalador NSIS — Python embutido, sem pré-requisitos.

JobTracker-Setup-1.0.0.exe

Instalação por usuário em %LOCALAPPDATA%\Programs\JobTracker. Dados em %APPDATA%\JobTracker. Atalhos de desktop e menu Iniciar opcionais.

Guia de instalação completo

Interface

Dashboard diário com timeline, controles de atividade e barra de progresso.

Acesse em http://localhost:5000 após iniciar o servidor.
