Track your hours without friction

A lightweight, local, self-hosted hour-tracking tool for agency professionals. Web UI at localhost:5000, JSON storage, no database.

Install Documentation
Python 3.10+ Flask Bootstrap 5 Linux Windows

Main features

Clean state machine

Start, pause, resume and stop activities. Pause time is subtracted automatically from total duration.

Daily dashboard

Visual timeline, shift-aware progress bar and target percentage. Only counts elapsed hours.

JSON storage

One file per month in data/YYYY-MM.json. No ORM, no DB. Easy to inspect by hand.

Configurable shifts

Define shifts per weekday with multiple blocks. Port, theme (light/dark/auto) and target also editable.

CSV/TSV export

Formatted for manual paste into iClips. Exports only completed activities in a date range.

Optional system tray

Pause, resume and stop from the system tray icon. Requires pystray (graceful fallback if missing).

Atomic writes

Every write uses temp file + fsync + os.replace(). Safe against crashes and power loss.

Light / dark theme

Automatic from the OS, or fixed. Preference persisted in localStorage.

pt-BR interface

UI strings in Brazilian Portuguese. Built around the daily time-logging flow of agency work.

Quick install

Linux / macOS

Requires Python 3.10+ and git.

git clone https://github.com/rafaelkrause/job_tracker.git
cd job_tracker
./install.sh
./job-tracker.sh

The script creates a .venv, installs dependencies and opens the browser.

Windows

Use the NSIS installer — embedded Python, no prerequisites.

JobTracker-Setup-1.0.0.exe

Per-user install at %LOCALAPPDATA%\Programs\JobTracker. Data in %APPDATA%\JobTracker. Desktop and Start Menu shortcuts optional.
