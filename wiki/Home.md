## Português

Ferramenta leve e local de rastreamento de horas, pensada para profissionais de agência. Roda como servidor web em `localhost:5000`, sem banco de dados, com armazenamento em arquivos JSON.

## English

Lightweight, local hour-tracking tool built for agency professionals. Runs as a web server at `localhost:5000`, no database, with JSON file storage.
